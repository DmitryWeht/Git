1.
На GitHub.com перейдите на главную страницу репозитория.

Над списком файлов щелкните  Код.

щелкните SSH и скоприуйте ссылку

Откройте GIT Bash.

Измените текущий рабочий каталог на расположение, где должен находиться клонированный каталог.

Введите git clone и вставьте URL-адрес, скопированный ранее.

$ git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY

2.
git init
git add
git commit -m 'new'
git remote add origin < SSH ссылка>
git push -u origin main

3.
git pull

4. 
Команда git status предоставляет информацию о текущем состоянии вашего локального репозитория Git.

Команда git log используется для просмотра истории коммитов в вашем локальном репозитории Git.

gКоманда git remote -v позволяет просмотреть список удаленных репозиториев, связанных с вашим локальным репозиторием Git, и их URL-адреса.
